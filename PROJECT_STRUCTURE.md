## 🔧 Scripts Folder (/scripts)

```
scripts/
└── fetchData.js                    → Main data fetching script
```

### What fetchData.js Does
1. Reads config.json
2. Fetches from GitHub API
3. Processes data:
   - Merges with config
   - Applies overrides
   - Filters hidden repos
   - Features marked repos
4. Generates JSON files:
   - profile.json
   - repos.json
   - starred.json

---

## ⚙️ GitHub Automation (.github)

```
.github/
└── workflows/
    └── update.yml                  → GitHub Actions workflow
```

### What update.yml Does
- Runs on schedule (weekly)
- Runs on push (optional)
- Executes fetchData.js
- Commits updated JSON
- Triggers Netlify rebuild

## 📋 Configuration System

### config.json Structure
```json
{
  "username": "your-github-username",     // REQUIRED
  "override": {                           // Optional overrides
    "name": "Your Name",
    "title": "Your Title",
    "bio": "Your bio",
    "email": "your@email.com",
    "location": "Your location",
    "linkedin": "URL",
    "twitter": "URL",
    "website": "URL"
  },
  "projects": {                           // Per-project customization
    "repo-name": {
      "featured": true,
      "demo": "URL",
      "live": "URL",
      "description": "Custom desc"
    }
  },
  "skills": {                             // Skill categorization
    "featured": ["Skill1", "Skill2"],
    "hidden": []
  },
  "hiddenRepos": [],                      // Hide certain repos
  "featuredRepos": []                     // Feature certain repos
}
```

## 🎨 Styling System

### File Organization
```
src/styles/
└── global.css                      → All global styles

tailwind.config.mjs                 → Color, font, theme config
postcss.config.mjs                  → CSS processing
```

### How Styling Works
1. Tailwind CSS classes in components
2. Custom utilities in global.css
3. Colors from tailwind.config.mjs
4. PostCSS processes everything
5. Final CSS is minified

### How to Change Colors
Edit `tailwind.config.mjs`:
```javascript
colors: {
  primary: {
    500: '#0ea5e9',  // Change this
  }
}
```

## 📈 How Files Are Generated

### When `npm run fetch-data` Runs

1. **Reads:** `config.json`
2. **Fetches:** GitHub API
3. **Processes:**
   - Merges GitHub + config data
   - Applies overrides
   - Filters repos
   - Extracts skills
4. **Generates:**
   - `public/data/profile.json` (350+ lines)
   - `public/data/repos.json` (varies)
   - `public/data/starred.json` (varies)
